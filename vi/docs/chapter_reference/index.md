# Tài liệu tham khảo

[1] Thomas H. Cormen, và cộng sự. Giới thiệu về thuật toán (Ấn bản thứ 3).

[2] Aditya Bhargava. Thuật toán Grokking: Hướng dẫn minh họa dành cho lập trình viên và những người tò mò khác (Ấn bản đầu tiên).

[3] Robert Sedgewick, và cộng sự. Thuật toán (Tái bản lần thứ 4).

[4] Yến Vi Mẫn. Cấu trúc dữ liệu (Phiên bản ngôn ngữ C).

[5] Đặng Junhui. Cấu trúc dữ liệu (Phiên bản ngôn ngữ C++, Phiên bản thứ ba).

[6] Mark Allen Weiss, do Chen Yue dịch. Cấu trúc dữ liệu và phân tích thuật toán trong Java (Ấn bản thứ ba).

[7] Trình Khiết. Cấu trúc dữ liệu bằng ngôn ngữ đơn giản.

[8] Vương Tranh. Vẻ đẹp của cấu trúc dữ liệu và thuật toán.

[9] Gayle Laakmann McDowell. Giải mã cuộc phỏng vấn viết mã: 189 câu hỏi và giải pháp lập trình (Ấn bản thứ 6).

[10] Aston Zhang và cộng sự. Đi sâu vào Học sâu.
